# Aerospace Alloy Strength Prediction using Machine Learning

## Overview

This project predicts the tensile strength of aerospace aluminum alloys using Machine Learning and Materials Informatics techniques.

The model uses alloy composition and heat-treatment parameters to estimate tensile strength and assist in material optimization.

## Problem Statement

Develop a machine learning model that predicts tensile strength based on:

- Magnesium (Mg)
- Copper (Cu)
- Zinc (Zn)
- Heat Treatment Temperature
- Aging Time

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Machine Learning Model

- Linear Regression

## Model Performance

- R² Score: 0.958
- RMSE: 4.658 MPa

## Results

The model successfully predicted tensile strength with high accuracy and proposed an optimized aerospace alloy composition.
